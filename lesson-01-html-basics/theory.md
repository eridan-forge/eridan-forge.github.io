# 📖 Теория к Уроку 1: HTML — скелет сайта

## Что такое HTML?
HTML (HyperText Markup Language) — это язык, на котором пишут каркас любого сайта.  
Браузер читает HTML-файл и показывает нам страницу с текстом, картинками, кнопками.

HTML состоит из **тэгов** (команд в угловых скобках).

---

## 🔧 Главные тэги (которые нужно знать)

| Тэг | Зачем нужен |
|-----|-------------|
| `<h1>` | Самый главный заголовок (один на странице) |
| `<h2>`, `<h3>` | Заголовки поменьше |
| `<p>` | Абзац текста |
| `<img>` | Вставить картинку |
| `<a>` | Ссылка на другой сайт |

---

## 🧱 Структура HTML-файла

```html
<!DOCTYPE html>     — объявляем, что это HTML
<html>              — всё, что внутри — страница
<head>              — настройки (не видно)
    <title>Заголовок вкладки</title>
</head>
<body>              — то, что видит пользователь
    <h1>Заголовок</h1>
    <p>Текст</p>
</body>
</html>
