## 🧠 Что такое событие (event)

Событие — это действие пользователя:
- нажатие на кнопку (`click`)
- движение ползунка (`input`)
- наведение мыши (`mouseenter`)

JavaScript может **следить** за событиями и выполнять код в ответ.

### Пример (простой)
```js
button.onclick = function() {
    alert('Кнопка нажата!')
}
```


🎛️ Что такое слайдер (range input)
В HTML есть специальный элемент для ползунков:

<input type="range" min="0" max="180" value="90">

min — минимальное значение

max — максимальное

value — текущее

Зачем это робототехнику?

Каждый палец механической руки может поворачиваться от 0° до 180°.

🧩 Как связать слайдер и JavaScript

Чтобы получить значение слайдера в реальном времени:

```js
slider.oninput = function() {
    let angle = slider.value
    console.log(angle)
}
```

🧱 Что такое DOM (Document Object Model)

DOM — это представление HTML-страницы в виде дерева объектов.

JavaScript может:

находить элементы (getElementById)

менять их содержимое (innerText)

менять стили (style.color)

🧠 Как работает лог (журнал действий)

Лог — это список последних команд.
Мы храним его в массиве и выводим на страницу.

```js
let logMessages = []

function addLog(text) {
    logMessages.unshift(text)   // добавить в начало
    if (logMessages.length > 10) logMessages.pop()   // не больше 10
    // показать на экране
}
```

🕹️ Что такое макрокоманда (например, «Махнуть рукой»)

Макрокоманда — это последовательность простых команд.

```js
function waveHand() {
    setServo(0, 30)
    setServo(1, 30)
    setTimeout(() => {
        setServo(0, 90)
        setServo(1, 90)
    }, 800)
}
```
