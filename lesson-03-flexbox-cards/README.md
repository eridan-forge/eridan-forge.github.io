## 🎯 Цель урока

Научиться создавать **сетку карточек** с помощью Flexbox и делать **адаптивный дизайн**, который красиво выглядит на любом экране.

---

## ✅ Что узнают ученики

- Что такое Flexbox и зачем он нужен
- Основные свойства: `display: flex`, `justify-content`, `align-items`, `gap`, `flex-wrap`
- Как сделать адаптивную сетку карточек
- Как создать карточку проекта (иконка, заголовок, описание, тег)
- Как добавить эффекты при наведении мыши (`:hover`)

## 🔧 Технологии

- **HTML5** — структура карточек
- **CSS3** — Flexbox, адаптив, псевдоклассы
- **JavaScript** — простой `alert` при клике на карточку (для интерактива)

## 🖼️ Как выглядит результат

Светлый фон с лёгкой текстурой, 4 карточки проектов:
- Механическая рука (Arduino)
- Панель управления (HTML/CSS/JS)
- Робот-машина (C++)
- Джойстик для робота (JS + Gamepad API)

При наведении карточка приподнимается и меняет цвет фона.  
При клике — показывает название проекта.

## 🏠 Домашнее задание

2. Изменить цвет акцента на свой
3. Сделать так, чтобы при клике на карточку открывался не `alert`, а ссылка на другой сайт
