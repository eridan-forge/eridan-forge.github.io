# 📖 Теория к Уроку 2: CSS — добавляем стиль

## Что такое CSS?
CSS (Cascading Style Sheets) — это язык, который отвечает за **внешний вид** сайта.

HTML — скелет (где голова, где руки).  
CSS — одежда (цвета, шрифты, отступы, тени).

## Как подключить CSS?
Есть 3 способа. Самый простой для начала — через тег `<style>` внутри `<head>`:

```html
<head>
    <style>
        body { background: black; }
    </style>
</head>
Основные CSS-свойства
Свойство	Зачем
color	цвет текста
background	цвет фона
padding	отступ ВНУТРИ элемента
margin	отступ СНАРУЖИ элемента
border-radius	скругление углов
text-align	выравнивание текста
font-size	размер шрифта
Блок и селекторы
body { } — стили для всей страницы

.card { } — стили для всех элементов с классом card

#id { } — для уникального элемента

Псевдокласс :hover
Меняет стиль при наведении мыши.

css
button:hover {
    background: darkred;
}
Итог
CSS делает сайт красивым, удобным и «дорогим» на вид.
